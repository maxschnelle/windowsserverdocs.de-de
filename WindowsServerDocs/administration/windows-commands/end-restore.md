# <a name="end-restore"></a>Wiederherstellung beenden



Beendet eine Wiederherstellungs Sitzung und gibt ein **postrestore** -Ereignis für beteiligte Writer aus.

## <a name="syntax"></a>Syntax

```
end restore
```
